Bezpečnostné a technické nálezy

Zoradené podľa závažnosti. Každý nález obsahuje konkrétny technický detail.

Kritické Žiadne HTTP security headers
Server neposiela Strict-Transport-Security, X-Frame-Options, X-Content-Type-Options, Content-Security-Policy, Referrer-Policy ani Permissions-Policy. Web je zraniteľný voči clickjacking, MIME sniffing a ďalším útokom.
Kritické Google Analytics kompletne nefunguje
Na webe sú 2 skripty Universal Analytics (UA-123265846-1, UA-67007434-5), ale UA bolo Google vypnuté v júli 2024. Žiadny GA4 tag neexistuje. PMD nemá žiadne analytické dáta minimálne 20 mesiacov.
Kritické WP login formulár na každej verejnej stránke
V sidebar widgete je na všetkých stránkach plný prihlasovací formulár s akciou na wp-login.php. Zbytočne zvyšuje útočnú plochu a umožňuje brute-force útoky z akejkoľvek stránky.
Kritické xmlrpc.php + pingback verejne prístupné
Endpoint xmlrpc.php je dostupný a <link rel="pingback"> je v hlavičke každej stránky. Riziko DDoS amplifikačného útoku a brute-force cez XML-RPC.
Vysoké Exponované verzie softvéru
Server header odhaľuje nginx/1.25.4, meta generator tagy prezrádzajú WordPress 6.9.4 a WooCommerce 10.6.1. Uľahčuje cielené útoky na známe zraniteľnosti.
Vysoké Prázdny robots.txt, chýba meta description a OG tagy
Súbor robots.txt má 0 bajtov — neblokuje crawling /wp-admin/. Homepage nemá <meta name="description">, Open Graph ani Twitter Card tagy. Žiadna vlastná XML sitemap (len základná WP sitemap). Chýba <h1> na homepage.
Vysoké Žiadna optimalizácia obrázkov
0% obrázkov vo formáte WebP. Na homepage 0 z 26 obrázkov má loading="lazy". Na stránke BUDÍK 2026 má 36 z 44 obrázkov (82%) prázdny alt text. Statické obrázky témy nemajú width/height atribúty — spôsobujú layout shift.
Vysoké 27 JavaScript súborov na každej stránke
Načítava sa jQuery + jQuery Migrate, dFlip 3D Flipbook (na každej stránke aj keď sa nepoužíva), Facebook SDK v10.0 (5 rokov stará), 2x reCAPTCHA (v2 + v3 súčasne s rôznymi kľúčmi), 2x mŕtve UA skripty, wp-polyfill a IE conditional comments.
Vysoké YouTube embed porušuje GDPR
Kaplnka Svetla používa štandardný YouTube embed namiesto youtube-nocookie.com. YouTube nastavuje tracking cookies bez predchádzajúceho súhlasu používateľa, čo je v rozpore s GDPR.
Stredné Chýbajúca funkčnosť oproti konkurencii
Oproti missio.org.uk a pontificalmissions.org chýba: vyhľadávanie na stránke, viacjazyčná verzia (min. EN), JSON-LD structured data, interaktívna mapa projektov, standalone newsletter prihlásenie, vizuálny impact dashboard so štatistikami. URL /aktuality/ a /blog/ vracajú 404 (funguje len /category/aktuality/).
Stredné UX problémy
Navigácia 6 úrovní hlboká. Stránka BUDÍK (udalosť) nemá registračný formulár. Stránka Adopcia seminaristov nemá online formulár (len IBAN číslo). Nefunkčná doména papezskemisijnediela.sk. Preklep "Kontakny" namiesto "Kontaktný". Nginx header wall: awww jisss. Viewport meta chýba initial-scale=1.
Stredné Zastaraná téma a technológie
Téma twentytwelve z roku 2012 s child témou. Float-based layout namiesto Flexbox/Grid. IE conditional comments (<!--[if IE 7]>) stále v kóde. FlexSlider 1.8 (neudržiavaný jQuery slider). Facebook SDK v10.0 namiesto aktuálnej v21.0.
